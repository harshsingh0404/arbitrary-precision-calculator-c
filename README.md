This project demonstrates the implementation of an Arbitrary Precision Calculator in C using linked lists.

📌 Overview

The Arbitrary Precision Calculator (ACP) is a C-based project that performs arithmetic operations on very large integers that exceed the limit of standard data types such as int or long.

To overcome this limitation, numbers are stored using a linked list data structure, where each node represents a single digit of the number. Arithmetic operations are performed digit by digit, making it possible to handle numbers of virtually unlimited size.

This project demonstrates practical implementation of data structures and algorithms in C programming.

🚀 Features

Handles very large numbers beyond built-in data type limits

Implements fundamental arithmetic operations:

Addition

Subtraction

Multiplication

Division

Uses linked lists for storing digits

Efficient handling of carry and borrow operations

Command-line based calculator

Programming Language: C
